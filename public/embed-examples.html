Basic HTML Embed:

<iframe src="http://localhost:3000/embed?lang=AR&color=%23e11d48" width="100%" height="400" frameborder="0" style="border-radius: 8px; box-shadow: 0 4px 6px rgba(0, 0, 0, 0.1);"> </iframe>

Responsive Embed:

<div style="position: relative; width: 100%; max-width: 400px; height: 400px;"> <iframe src="http://localhost:3000/embed?lang=EN&color=%233b82f6" style="position: absolute; top: 0; left: 0; width: 100%; height: 100%; border: none; border-radius: 8px;"> </iframe> </div>

WordPress Shortcode Style:

[portfolio_card lang="AR" color="#10b981"]

Available Parameters:

Parameter Description Default Value Example
lang Language (AR for Arabic, EN for English) AR lang=EN
color Primary color (hex code, URL encoded) #3b82f6 color=%23e11d48

Color Encoding:

Remember to URL encode the # symbol in hex colors:

  • #3b82f6 becomes %233b82f6
  • #e11d48 becomes %23e11d48
  • #10b981 becomes %2310b981
