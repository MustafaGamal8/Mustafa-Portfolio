Troubleshooting

If the iframes show "This page can't be displayed in a frame" or similar errors:

  • ✅ Check that the Next.js server is running on localhost:3000
  • ✅ Verify X-Frame-Options headers are set to ALLOWALL for /embed routes
  • ✅ Ensure Content-Security-Policy allows frame-ancestors
  • ✅ Check browser console for any errors
